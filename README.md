📊 Análise do Ensino de Ciências no Ensino Fundamental
📌 Sobre o projeto

Este projeto analisa a percepção de alunos do 6º, 8º e 9º ano sobre o ensino de Ciências.

🎯 Objetivos
- Identificar dificuldades no aprendizado
- Avaliar o interesse dos alunos
- Analisar o conforto em participar das aulas

🛠️ Ferramentas utilizadas
Python
Pandas
Matplotlib

#Dados da planilha não estão disponíveis.
